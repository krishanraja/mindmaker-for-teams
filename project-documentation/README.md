## Terminology Standards

- **Alignment Sprint**: The core 3-hour workshop format
- **Battle Test**: Segments 4-6 where teams test decisions through simulation
- **Decision Framework**: The output artifact showing how the team makes decisions
- **Facilitator**: Workshop leader using the platform
- **Participant**: Executive team member attending workshop
- **Workshop Session**: A single run of the alignment sprint
- **Unified Profile**: Cross-tool user identity system
- **Segment**: One of seven phases in the workshop (Mythbuster, Bottleneck Board, Effortless Enterprise, Simulation Lab, Strategy Addendum, Pilot Charter, Provocation)
